## :information_source: How To Use

```bash
# Clone this repository
$ git clone https://github.com/victor-vas/ecommerce-react-app search-github-users

# Go into the repository
$ cd search-github-users

# Install dependencies
$ yarn

# Run the app
$ yarn start
```
